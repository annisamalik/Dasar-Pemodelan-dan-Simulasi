# Dasar-Pemodelan-dan-Simulasi

Ini adalah tugas pemrograman yang saya buat untuk memenuhi mata kuliah Dasar Pemodelan dan Simulasi

##PENGARUH RESISTENSI LINEAR TERHADAP BOLA MEMANTUL
Pemodelan ini dilatarbelakangi oleh ketertarikan penulis terhadap gerak bola memantul. Bola memantul jika diamati membentuk pola lintasan gerak peluru yang iteratif. Penulis ingin mencari tahu pengaruh resistensi linear terhadap bola memantul serta membandingkan hasil solusi analitik dan solusi numerik. Program yang digunakan untuk memodelkan bola memantul adalah Octave. Hasil menunjukkan bahwa selisih hasil solusi numerik dan analitik berbeanding lurus dengan Î”t. Simulasi bola memantul dengan Hukum Stokes juga menunjukkan bahwa lintasan tidak membentuk parabola yang simetris dan kecepatan semakin melambat karena adanya gaya resistensi linear.
